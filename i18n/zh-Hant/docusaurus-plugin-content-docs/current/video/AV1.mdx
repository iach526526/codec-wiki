# AV1

AV1 是一種買斷式授權 (royalty-free)的視頻壓縮格式，旨在繼承 [VP9](../video/VP9.mdx) 的地位。目前，它與 [VP9](../video/VP9.mdx)、[VVC](../video/VVC.mdx) 和 [HEVC](../video/HEVC.mdx) 競爭。相較於 VP9，AV1 的計算複雜度更高，但由於成熟高效的 dav1d AV1 解碼器，解碼速度相當快。此外，各種消費硬體裝置也已經支援 AV1 的硬體加速解碼，所有相關設備詳見 [維基百科](https://en.wikipedia.org/wiki/AV1#Hardware)。其中包括現代的 Intel、AMD 和 Nvidia 的整合式與獨立 GPU，Google Pixel 系列使用的 Tensor SoC，Apple A17 Pro（搭載於 iPhone 15 Pro 系列），以及現代的 Mediatek 和 Qualcomm 晶片。目前，YouTube 正在逐步將其視頻轉換為使用 AV1 格式。

現今有多種可行的 AV1 編碼解決方案。最優秀、最普及且免費的三大方法是 [aomenc](../encoders/aomenc.mdx)、[SVT-AV1](../encoders/SVT-AV1.mdx) 和 [rav1e](../encoders/rav1e.mdx)。
