# UT Video Codec Suite

:::danger 我們需要你！
此部分需要更多內容。如果你有意協助，請參考我們的 [貢獻指南](../contribution-guide.mdx) 並開始貢獻！
:::

UT Video Codec Suite 是一款快速、無失真的影片編解碼器，由 梅澤威志（Takeshi Umezawa）開發，並以 GNU 通用公共許可證（GPL）免費發布。UT Video 的算法基於霍夫曼編碼（Huffman code）。

UT Video 是為了替代 HuffYUV 而開發的，目的是實現更好的壓縮效果。它可以處理多種色彩空間，例如 YUV422 (ULY2)、RGB (ULRG)、RGBA (ULRA)，以及最近新增的 YUV420 (ULY0)。

該編解碼器有 x86 和 x64 版本，由於其支援多執行緒，該編解碼器還能夠實時編碼 HDTV 素材。此外，由於該編解碼器高度依賴於 SSE2 指令集進行速度優化，因此需要支援 SSE2 的硬體平台。

UT Video Codec Suite 提供了多種預測模式，可以通過 [FFmpeg](/docs/utilities/ffmpeg) 使用：
- 無預測模式
- 左鄰預測（對整個片段進行連續預測）
- 梯度預測
- 中值預測

你可以使用以下 FFmpeg 命令來編碼 utvideo：
```bash
ffmpeg -i [輸入檔案] -c:v utvideo -pred [0,1,2,3] [輸出檔案]
```
