# H.265

H.265，亦稱高效能視訊編碼（ High-Efficiency Video Coding ，簡稱 HEVC），是一項視訊壓縮標準，旨在以更小的檔案大小提供更高畫質的影片。
作為 [H.264](../video/AVC.mdx) 編碼的後繼者，它應運而生，以應對對高效且高畫質視訊壓縮技術日益增長的需求，尤其是在 4K 內容的情境下。然而，這項編碼標準受限於專利權費用。
