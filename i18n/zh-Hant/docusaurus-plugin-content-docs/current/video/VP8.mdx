# VP8

VP8 是由 On2 Technologies 於 2008 年開發的影片壓縮格式，並於 2010 年由 Google 釋出為免版稅的編解碼器。VP8 的效率與 [AVC](../video/AVC.mdx) 相當，
在免版稅編解碼器領域中曾是一個重要的角色，並被廣泛應用於網絡影片傳輸等多種用途。VP8 是 [VP9](../video/VP9.mdx) 和 [AV1](../video/AV1.mdx) 的前身，後者在影片壓縮效率上進一步提升。
然而，VP8 也受到批評，因其規範被認為凌亂且不完整。
