# VP9

:::danger 我們需要你！
此部分需要更多內容。如果你有意協助，請參考我們的 [貢獻指南](../contribution-guide.mdx) 並開始貢獻！

VP9 是一種免版稅的影片壓縮格式，因為多年來成為 YouTube 的首選編碼器而聞名。它與 [HEVC](../video/HEVC.mdx) 和 [AVC](../video/AVC.mdx) 競爭。儘管在視覺品質上稍遜於 HEVC，但 VP9 是一種計算上更簡單的編碼器，因此在解碼時對系統資源的需求較低。如果你對 VP9 編碼感興趣，請參閱 [vpxenc](../encoders/vpxenc.mdx) 或 [SVT-VP9](../encoders/SVT-VP9.mdx) 章節。
