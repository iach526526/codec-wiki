# Theora

Theora 是一款於 2004 年首次發布的遺留影片編碼器。它源自 VP3，屬於與 [VP8](./VP8.mdx)、[VP9](./VP9.mdx) 和 [AV1](./AV1.mdx) 同一系列的編碼器家族。

## 編碼
Theora 通常會包含在 ogg (`.ogg` 或 `.ogv`) 容器中，並搭配 [vorbis](../audio/Vorbis.mdx) 音頻。
