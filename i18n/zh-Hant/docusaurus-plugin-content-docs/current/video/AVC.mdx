# H.264

H.264，也被稱為 AVC（Advanced Video Coding，先進視訊編碼），是一種在多媒體編碼技術中具有重要地位的視訊壓縮標準。H.264 通過提供遠比其前任更高效的壓縮技術，
徹底改變了視訊編碼領域，並自此成為幾乎通用的視訊壓縮標準。H.264 的歷史充滿了持續的改進和廣泛的採用，特別是在高度優化的 [x264](../encoders/x264.mdx) 視訊編碼器中得到了充分展現。
儘管 H.264 已經問世超過十年，但它今天依然具有重要意義。雖然較新的編碼標準如 [H.265](../video/HEVC.mdx)、[VP9](../video/VP9.mdx) 和 [AV1](../video/AV1.mdx) 旨在提供比 H.264 
更高效的壓縮效果，但目前它們還未能達到 H.264 的普及程度。選擇使用哪種編碼標準，通常取決於應用的具體需求、相容性與支援情況，以及壓縮效率與計算複雜度之間的平衡。
