# VC-1

VC-1 是由微軟於 2006 年推出的一款影片編碼器，主要目的是與 [AVC](../video/AVC.mdx) 競爭。

## 編碼
VC-1 可以封裝在 `.mp4`、`.mkv` 和 `.avi` 容器中。
（待補充）

## 解碼
VC-1 可通過 [FFmpeg](../utilities/ffmpeg.mdx)、[VLC](../video-players.mdx)、[MPV](../video-players.mdx) 進行解碼，也可以在支援藍光光碟的設備上解碼。
