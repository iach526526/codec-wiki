# ECM

ECM (Enhanced Compression Model) is a video codec developed by the Fraunhofer Heinrich-Hertz-Institute. It is purportedly designed to succeed H.266, or [VVC](../video/VVC.mdx).

The [Git repo](https://vcgit.hhi.fraunhofer.de/ecm/ECM) provides a bit more information:

```md
# ECM 參考軟體

這個軟體套件是用於增強壓縮模型（Enhanced Compression Model, ECM）的參考軟體。該軟體包含編碼器和解碼器功能。參考軟體對於使用視訊編碼標準的用戶來說非常有幫助，能夠協助他們建立並測試標準的符合性和互操作性，同時也可以用於教育用戶和展示測試模型的功能。

這套軟體由 ITU-T 視訊編碼專家組（VCEG，ITU-T 第16研究組的第6問題）和 ISO/IEC 動態影像專家組（MPEG，ISO/IEC 聯合技術委員會第29分委會第11工作組）共同開發。

```
除了這些概述之外，有關 ECM 的構建指令已在倉庫的 README 中提供。目前關於 ECM 的其他資訊不多，只知道它很可能會成為 H.267 的候選標準。
