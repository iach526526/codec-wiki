# WebVTT

WebVTT（Web Video Text Tracks）是網頁上的字幕格式。它與 [HTML `<track>` 元素](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/track) 一起使用，或嵌入到 webm 容器中。

## 結構

WebVTT 是一種簡單的基於文本的格式，基於 [SRT](./SRT.mdx)。所有 WebVTT 檔案都以字符串 `WEBVTT` 開頭，之後是可選的文字，再接著是兩個換行符。這樣開始的部分就是我們感興趣的資料部分。

### Cue

WebVTT 檔案基本上是由一系列 cue（提示）組成的。它們可以有一行 ID，然後必須有一行指定提示的顯示時間範圍，格式如下：`STARTTIME --> ENDTIME [optional settings go here]`，接著是顯示的文本。這些文本可以包含一些 HTML 類似的格式化。欲了解更多資訊，請參見 [文檔](https://developer.mozilla.org/en-US/docs/Web/API/WebVTT_API#webvtt_cues)。

## 範例

```plaintext
WEBVTT

00:01.000 --> 00:04.000
- Never drink liquid nitrogen.

00:05.000 --> 00:09.000
- It will perforate your stomach.
- You could die.
```
