SubStation Alpha（SSA），也被稱為 Advanced Substation Alpha（ASS）v4+，是一種字幕格式。最初由同名 Windows 程式所使用，主要針對卡拉 OK 和動漫社群。由於其相較於其他格式的進階樣式功能，使其在發行群體中頗受歡迎。

# 概覽
SSA 是一種「純」文本格式，在 ASS 中支援 Unicode。它既可以保留為副檔名為 .ssa 或 .ass 的純文本檔案，也可以被合併進 Matroska (.mkv) 或 AVI (.avi) 檔案。原版的 SubStation Alpha 軟體已經成為被遺棄的舊版軟體。，但包括 [VLC](../video-players.mdx)、[MPV](../video-players.mdx) 和 [FFmpeg](../utilities/ffmpeg.mdx) 在內的多種媒體製作、混流和播放軟體都支援 SSA 格式。

## 格式
SSA 使用「腳本」一詞來指代與影片同步的字幕。以字符 ';' 開頭的行用來標記註解。
### ASS（SSA v4+）標頭
```
[Script Info]
; 這是一個 Advanced Sub Station Alpha v4+ 腳本。
; 如需 Sub Station Alpha 資訊和下載，
; 請訪問 http://www.eswat.demon.co.uk/
; 或電郵至 kotus@eswat.demon.co.uk
; 
; Advanced Sub Station Alpha 腳本格式由 #Anime-Fansubs@EfNET 開發
; http://www.anime-fansubs.org
; 
; 如需其他資訊和下載，請訪問 http://vobsub.edensrising.com/
; 或電郵至 gabest@freemail.hu
; 
; 注意：此檔案由 Subresync 保存。
; 
ScriptType: v4.00+
Collisions: Normal
PlayResX: 384
PlayResY: 288
Timer: 100.0000

[V4+ Styles]
Format: Name, Fontname, Fontsize, PrimaryColour, SecondaryColour, OutlineColour, BackColour, Bold, Italic, Underline, StrikeOut, ScaleX, ScaleY, Spacing, Angle, BorderStyle, Outline, Shadow, Alignment, MarginL, MarginR, MarginV, Encoding
Style: Default,Tahoma,16,&H00000000,&H00ffffff,&H00ffffff,&H00c0c0c0,-1,0,0,0,100,100,0,0.00,1,2,3,2,20,20,20,1

[Events]
Format: Layer, Start, End, Style, Actor, MarginL, MarginR, MarginV, Effect, Text
Dialogue: 0,0:01:41.70,0:01:46.84,Default,,0000,0000,0000,,Le rugissement des larmes !\NTu es mon ami.
Dialogue: 0,0:02:00.99,0:02:02.87,Default,,0000,0000,0000,,Est-ce vraiment Naruto ?
```
