# VTM

:::danger 我們需要你！
此部分需要更多內容。如果你有意協助，請參考我們的 [貢獻指南](../contribution-guide.mdx) 並開始貢獻！

VTM 是原始的 [H.266](../video/VVC.mdx)（更廣為人知的 VVC）參考編碼器，它與 [VVenC](../encoders/VVenC.mdx) 等其他編碼器進行競爭。如今，它與其他 MPEG 參考編碼器如 [HM](../encoders/HM.mdx) 和 [JM](../encoders/JM.mdx) 一起，以其高冷門度而聞名，這是因為這些編碼器的使用複雜性和速度劣勢。然而，鑑於目前 VVC 編碼面臨的困難，VTM 可能更為實用，無論你選擇哪種編碼實現。與 JM 相比，x264 的效率更高。
