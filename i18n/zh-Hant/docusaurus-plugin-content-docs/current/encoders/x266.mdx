# x266

x266 是一款即將推出的 [H.266 / VVC](/docs/video/VVC) 軟體編碼器。目前關於這款編碼器的資訊還非常有限，只知道它仍在 [MulticoreWare](https://multicorewareinc.com) 的開發階段，並且有一個專門的 FAQ 頁面，該頁面位於 [2023年1月31日網絡研討會 FAQ](https://multicorewareinc.com/faq-webinar/x266-faq.html) 之後。

根據他們的 FAQ，H2 2023 是他們對 v1.0 公開發行的「大致預估」時間，但截至 2024 年 3 月 29 日，尚未有任何進一步的更新。
