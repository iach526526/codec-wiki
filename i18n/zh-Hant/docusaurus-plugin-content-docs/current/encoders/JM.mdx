# JM

:::danger 我們需要你！
此部分需要更多內容。如果你有意協助，請參考我們的 [貢獻指南](../contribution-guide.mdx) 並開始貢獻！
:::

JM 是最早的 [H.264](../video/AVC.mdx) 參考編碼器，比 [x264](../encoders/x264.mdx) 等後來的選擇更早問世。隨著 x264 的高效能與感知驅動的發展，它與其他 MPEG 參考編碼器如 [HM](../encoders/HM.mdx) 和 [VTM](../encoders/VTM.mdx) 一樣，由於其複雜的使用方式與速度的劣勢，成為了極少數使用的小眾工具。一般而言，x264 的效率要高於 JM。
