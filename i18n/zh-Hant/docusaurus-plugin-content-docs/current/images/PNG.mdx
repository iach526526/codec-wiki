### Portable Network Graphics (PNG)

PNG 是一種免費的無損圖像檔案格式，於 1996 年發佈。它被創建為 [GIF](./GIF.mdx) 的替代品，當時 GIF 是一種專有格式。PNG 在 2008 年通過 APNG 的發布獲得了類似 GIF 的動畫支援，現在所有主流網頁瀏覽器均支援 APNG。
