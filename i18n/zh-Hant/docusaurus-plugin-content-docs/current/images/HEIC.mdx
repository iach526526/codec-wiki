# HEIC

HEIC（High Efficiency Image Format）是一種較新的圖像編碼格式，旨在提供比傳統圖像格式如 [JPEG](./JPEG.mdx) 更優的壓縮效果和性能。HEIC 檔案內部使用 [HEVC](../video/HEVC.mdx) 編碼，這意味著該格式並非免買斷式授權。雖然這限制了其在網絡上的普及，但許多現代設備，包括整個 Apple 生態系統，都支援此格式。iPhone 預設使用 HDR HEIC 照片，利用 iPhone 的 HEVC 硬體影片編碼器來捕捉這些圖像。一些 Android 手機也能拍攝 HEIC 圖像，但這些圖像通常是從 JPEG 轉碼而來。HEIC 在很大程度上被 [AVIF](./AVIF.mdx) 超越，後者使用相同的容器來儲存 [AV1](../video/AV1.mdx) 壓縮的圖像。
