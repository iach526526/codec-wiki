# WebP

WebP 是 Google 於 2010 年首次發布的一種免費圖像檔案格式。它包括兩種主要的「模式」：一種是源於 [VP8](../video/VP8.mdx) 影片編解碼器的有損模式，另一種是 2011 年新增的創新無損模式。

## 編碼

### 使用 libwebp
libwebp 支援多種輸入格式，包括 WebP、JPEG、PNG、PNM（PGM、PPM、PAM）和 TIFF，並提供一個品質參數（-q），數值範圍從 0 到 100，0 代表最低品質、最小檔案，100 則代表最高品質、最大檔案。如果需要無損壓縮模式，你應該使用 -z 參數來指定編碼效率，其數值範圍從 0 到 9，0 表示最快編碼但檔案較大，9 則表示最慢編碼但檔案最小。
```bash
cwebp example.png -q 75 -o example.webp
```

## 解碼

```bash
dwebp example.webp -o example.png
```
