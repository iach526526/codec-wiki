# WebP

WebP 是 Google 於 2010 年首次發布的一種免費圖像檔案格式。它包括兩種主要的“模式”：一種是源於 [VP8](../video/VP8.mdx) 影片編解碼器的有損模式，另一種是 2011 年新增的創新無損模式。

## 編碼

### 使用 libwebp
libwebp 支援 WebP、JPEG、PNG、PNM（PGM、PPM、PAM）、TIFF 作為輸入格式，並且可以設定品質參數（`-q`），範圍從 0（最低品質，檔案最小）到 100（最高品質，檔案最大）。如果需要使用無損模式，則需要使用 `-z` 參數，其值表示編碼的努力程度，範圍從 0（最快編碼，檔案最大）到 9（最慢編碼，檔案最小）。

```bash
cwebp example.png -q 75 -o example.webp
```

## 解碼

```bash
dwebp example.webp -o example.png
```
