# 無損與有損音頻壓縮介紹

:::info 維護中

:::

數位音頻是指以數位形式記錄或轉換的聲音。要了解數位音頻，掌握一些基本概念是關鍵的，包括 **取樣**、**奈奎斯特頻率(Nyquist Frequency)** 和 **奈奎斯特–夏農取樣定理 (Nyquist-Shannon Sampling Theorem)**。

## 取樣與奈奎斯特頻率

**取樣** 是將連續的類比音頻信號轉換為離散數位信號的過程，通過在均勻的時間間隔內測量音頻信號的振幅來完成。這個測量的頻率稱為 *取樣率*，通常以每秒取樣次數或赫茲（Hz）來表示。例如，音頻 CD 的取樣率為 44,100 Hz，意味著每秒取樣 44,100 次。

為了準確地表示一個波形，你需要每個週期至少進行兩次取樣：一次捕捉波峰，一次捕捉波谷。如果取樣頻率低於每個週期兩次，你將無法準確區分不同的頻率；這就是 **奈奎斯特頻率** 的由來。**奈奎斯特頻率** 是 *離散信號處理系統取樣率的一半*。對於給定的取樣率，奈奎斯特頻率是可以準確取樣的最高頻率，而不會引入像混疊這樣的錯誤。例如，取樣率為 48,000 Hz 時，奈奎斯特頻率是 24,000 Hz。捕捉超過奈奎斯特頻率的頻率會產生 **混疊** 伪影，這會導致高頻成分在取樣信號中顯示為低頻，從而扭曲訊息。

**奈奎斯特–夏農取樣定理** 表明，為了避免混疊，取樣率必須至少是信號中最高頻率的兩倍。這一定理對於確保數位音頻能夠忠實於原始類比音頻至關重要。
## 無損音頻壓縮

無損壓縮的主要優勢在於能夠保持音質，這使得它非常適合用於專業音頻製作、檔案保存以及需要高音質的場合。然而，無損檔案的尺寸比有損檔案要大很多，這可能會對儲存和（尤其是）串流造成挑戰。

[FLAC](./FLAC.mdx)、[WavPack](./WavPack.mdx) 和 [ALAC](./ALAC.mdx) 是一些常見的無損音頻編解碼器，你很可能會在實際使用中遇到這些編解碼器。

## 有損音頻壓縮

有損壓縮的主要優點是可以顯著減少檔案大小，這使它非常適合用於串流、可攜設備和儲存空間有限的場合。然而，這樣的壓縮方式可能會導致音質損失，在精確的聽音環境中這種損失可能會變得明顯。

:::note 有損壓縮
如果你希望深入了解有損壓縮的相關內容，可以參考維基中的 [有損壓縮](../introduction/lossy.mdx) 條目。
:::

[MP3](./MP3.mdx)、[AAC](./AAC.mdx)、[Vorbis](./Vorbis.mdx) 和 [Opus](./Opus.mdx) 是一些常見的有損音頻編解碼器，你在實際使用中可能會遇到這些編解碼器。
