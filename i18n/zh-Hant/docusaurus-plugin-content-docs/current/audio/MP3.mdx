# MP3

MP3，即 MPEG-1 Audio Layer III 或 MPEG-2 Audio Layer III，是一種數字音頻編碼格式。它主要由德國的弗勞恩霍夫學會（Fraunhofer Society）開發，由卡爾海因茨·布蘭登堡（Karlheinz Brandenburg）領導，並得到其他國家的數字科學家支援。

MP3 定義在兩個 ISO/IEC 規範系列中：MPEG-1: 11172-3 和 MPEG-2: 13818-32。它使用有損壓縮，通常可以使檔案大小比未壓縮音頻大幅減少。

有損 MP3 壓縮的工作原理是通過嘗試減少（或近似）某些聲音組件的精確度，這些組件可能被認為（根據某些 [心理聲學](../introduction/terminology.mdx#perceputal--psychovisual--psychoacoustic) 分析）超出了大多數人類的聽覺能力，並儲存這些更顯著頻段的系數。

與 CD 音質的數字音頻相比，MP3 壓縮通常可以實現 75% 至 95% 的尺寸減少。例如，一個以 128 kbit/s 的固定位元率編碼的 MP3 檔案，其大小大約為原 CD 音頻的 9%。MP3 音頻在 320kb/s 的位元率下被認為是透明的。

儘管 MP3 格式於 1993 年定稿（1995 年進行修改以支援較低的取樣率和位元率），但今天仍然可以看到大量的 MP3 檔案。這是因為 MP3 可以被認為是第一個廣泛採用的音頻格式，能夠將高品質音頻壓縮到相對較小的檔案大小。雖然與更現代的格式如 [Opus](../audio/Opus.mdx) 和 [AAC](../audio/AAC.mdx) 相比，MP3 可能看起來不那麼引人注目，但它仍然得到許多設備和軟體的廣泛支援，並確實為現代編解碼器留下了強大的遺產。
