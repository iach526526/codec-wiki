# 影片播放器

市面上有許多不同的影片播放器，以下是幾款推薦的選擇：

## MPV

**MPV** 是一款開源且輕量的媒體播放器。它主要作為命令行應用程式，界面非常簡約。不過，也有許多前端界面可供使用，提供更完整的圖形用戶介面，如 Celluloid 和 IINA。MPV 支援廣泛的編解碼器和容器格式。

## VLC

**VLC** 是一款開源的媒體播放器和工具包，支援幾乎所有常見的影音編解碼器。

所有平台的 VLC 下載說明可以在 [VideoLAN 的網站](https://www.videolan.org/vlc/#download) 上找到。

## MPC-HC

**Media Player Classic - Home Cinema (MPC-HC)** 是一款專為 Windows 7、8、8.1、10 和 11 操作系統設計的開源媒體播放器。它以簡潔和高效聞名，為用戶提供了一個輕便且友好的平台來播放各種音訊和視頻格式。其輕量化設計確保了流暢的播放，並且不會消耗過多的系統資源。雖然官方的開發於 2017 年已經停止，但由社群維護的 MPC-HC 仍在 GitHub 上持續提供更新和支援。

由社群維護的最新穩定版和測試版可以在 [clsid2 的 MPC-HC 倉庫](https://github.com/clsid2/mpc-hc/releases) 下載。
