# Brotli

Brotli 由 Google 於 2013 年底發布，並廣泛應用於 Web 上的內容傳遞。它是 `.woff2` Web 開放字型格式的核心部分，允許網頁字型在發送給用戶時更小。雖然 `.tar.br` Brotli 壓縮檔案不像 [gzip](../data/gzip.mdx) 或 [xz](../data/xz.mdx) 那樣常見地被傳遞，但這樣的檔案不在廣泛使用的範圍內並不奇怪。Brotli 在網絡上的兼容性幾乎是普遍的，支援率高達 96%。

Brotli 基於 LZ77 和 Huffman 編碼，與 ZIP 類似。它還使用上下文建模，允許在同一區塊內使用多個 Huffman 樹來處理相同字母表。這意味著，根據壓縮資料的上下文，尤其是當資料包含多種不同類型的內容時，可以更有效地進行壓縮。

Brotli 由 Jyrki Alakuijala 共同編著和部分開發，他還參與了 [JPEG-XL](../images/JXL.mdx) 和高效 [JPEG](../images/JPEG.mdx) 編碼器 jpegli 的開發。JPEG-XL 的元資料訊息通常會進行 Brotli 壓縮。
