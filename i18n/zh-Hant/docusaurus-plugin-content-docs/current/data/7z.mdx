# 7-zip (7z)

7-zip（7z）是一種支援多種資料壓縮、加密和預處理演算法的檔案格式。它由 7-Zip 壓縮工具創建，這是一款免費且開源的軟體，能處理多種資料壓縮格式，包括與 7z 類似的 [XZ](../data/xz.mdx) 格式。

7-zip 格式相對於流行的 [ZIP](../data/zip.mdx) 格式具有一些顯著優勢：
- 7-zip 工具能夠比 ZIP 格式壓縮檔案「提高 30-70%」的效率，即使 ZIP 格式也有高效的編碼器。它主要使用 LZMA 和 LZMA2 演算法，這些比 DEFLATE 更現代，通常壓縮效果更佳。
- 7-zip 支援使用 AES-256 加密檔案，並使用用戶提供的密碼。AES-256 的安全性比 ZIP 常用的 ZipCrypto 加密更高。
- 7-zip 支援大小達 16 Exabytes 的檔案，而傳統 ZIP 有 4 GB 的限制（ZIP64 目前常用，不受這一限制，因此現在這點不那麼相關）。7-zip 還支援各種預處理過濾器，能改善某些類型資料的壓縮效果，如可執行檔和二進位檔案。

然而，7-zip 也有一些缺點和限制：
- 7-zip 的支援程度不如 ZIP 廣泛，某些軟體和平台可能需要安裝額外的程式或擴充套件來打開或解壓 7z 檔案。
- 壓縮或解壓速度較慢：與 ZIP 相比，7-zip 壓縮檔案可能需要更長的時間。雖然 7-zip 工具在解碼時有效地進行了平行處理，但這僅影響實際時間，而不是用戶時間，意味著解壓過程仍可能比 ZIP 更耗時。
- 7-zip 沒有內建修復損壞檔案的機制。用戶可能需要使用第三方工具或備份副本來恢復資料。

7z 檔案在 macOS 和許多 Linux 發行版中原生支援。
