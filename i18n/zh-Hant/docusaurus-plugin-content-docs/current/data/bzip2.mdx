# bzip2

:::danger 我們需要你！
此部分需要更多內容。如果你有意協助，請參考我們的 [貢獻指南](../contribution-guide.mdx) 並開始貢獻！
:::

bzip2 是一款開源的檔案壓縮格式和工具。它的壓縮效率略優於 [zip](./zip.mdx)，但不如基於 LZMA 的格式，如 [xz](./xz.mdx) 和 [7z](./7z.mdx)。bzip2 不能直接壓縮多個檔案，
使用 bzip2 壓縮多個檔案時，應該先將檔案整合成 [tarball](./tar.mdx)。
