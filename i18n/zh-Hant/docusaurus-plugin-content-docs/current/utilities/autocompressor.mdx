# Autocompressor

[Autocompressor](https://autocompressor.net/) 
是由 Auto-Rez Media Technologies 提供的免費線上媒體壓縮工具，
能將視頻、圖片、音頻和 GIF 壓縮至指定的檔案大小。它根據上傳的檔案類型，
採用不同的處理方式，確保結果在功能上與原檔案相符。例如，視頻會轉換為視頻格式，
音頻檔案轉換為音頻檔案，圖片轉換為圖片，而動畫則轉換為動畫。

網站上提供的訊息：
> 使用我們的免費線上壓縮工具，可以快速輕鬆地縮小影片、圖片、音頻或 GIF 的檔案大小。
無論你需要將檔案縮小至 25MB 還是其他特定大小，我們的工具都提供了簡單直觀的解決方案。
我們還會為 Discord 創建完美壓縮的表情符號和貼紙。使用鏈接模式，你可以立即將 YouTube 影片和其他網站的影片壓縮至 25MB。
我們先進的 AUTO-REZ™ 技術能夠在給定檔案大小下實現比其他網站更高的品質。Autocompressor 將根據輸入格式和選擇的兼容性類別自動選擇輸出格式。

Autocompressor 未來計劃提供一個高級服務，其成本將根據操作過程中使用的 CPU 分鐘數以及壓縮工作量層級來決定。例如，將一個典型的 4 分鐘 YouTube 影片壓縮到 8MB 的 MP4-Normal 格式約需 5 CPU 分鐘。

用戶可以創建帳戶以獲得 1800 CPU 分鐘免費使用。網站通過使用 BCrypt 雜湊儲存密碼來確保用戶資料的高度安全性。

你可以在他們的 [關於頁面](https://autocompressor.net/about) 了解更多。
