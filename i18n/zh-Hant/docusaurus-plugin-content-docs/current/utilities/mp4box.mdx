# MP4Box

:::danger Help Wanted
This section is in need of contributions. If you believe you can help, please see our [Contribution Guide](../contribution-guide.mdx) to get started as a contributor!
:::

MP4Box is a software for multi-purpose MP4 file manipulation. Part of the free and open-source multimedia framework called [GPAC](https://github.com/gpac/gpac) for working with the MP4/ISOBMFF standard.
