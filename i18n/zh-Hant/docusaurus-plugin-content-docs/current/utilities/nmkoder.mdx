[NMKODER](https://github.com/n00mkrad/nmkoder) 是一款專為 **Windows** 平台開發的免費且開源的圖形介面軟體，由 [N00MKRAD](https://github.com/n00mkrad) 開發。它以 FFmpeg、FFprobe 和 Av1an 作為核心技術，提供影片編碼、封裝和分析功能，包括串接、指標計算和位元率圖表繪製。大多數使用者選擇這款軟體主要是為了使用 [Av1an](/docs/utilities/av1an) 的相關功能。

此軟體由 C# 語言撰寫，並使用 WinForms 架構來構建。

:::danger Abandonware
NMKODER 已多年未更新，不建議繼續使用。不過，你可以嘗試替換軟體中提供的二進制檔案（如 aomenc.exe 等），從技術上「更新」這個軟體。
:::

## 安裝指南

你可以從 GitHub 的 [releases](https://github.com/n00mkrad/nmkoder/releases) 頁面下載 7z 格式的壓縮包。下載後將其解壓至任意位置，然後進入解壓後的檔案夾，點擊其中的 exe 檔案啟動軟體。

這款軟體是便攜版的，因此你可以將整個檔案夾分享給他人，使用時不會遇到任何問題。

## 使用方法

NMKODER 的介面設計簡單明瞭，每個功能對應一個標籤頁。左側提供了一個拖放區域，方便用戶直接拖放影片檔案進行操作。

- 「File List」是導入檔案的列表。
- 「Track List」顯示音影片軌道。
- 「Quick Convert」標籤頁用於通過 FFmpeg 進行快速轉換。
- 「AV1AN」標籤頁提供了與名稱對應的功能。
- 「Utilities」標籤頁則提供了一些實用工具，如色彩空間或 HDR 元資料的轉移、指標計算等。

完成設定後，點擊「Start」按鈕即可開始編碼。
