# Av1an Command Generator

Av1an Command Generator 是一款簡單易用的工具，用於生成 AV1 編碼的 Av1an 命令。
它使用 Zig 這門程式語言，功能類似於 [rAV1ator CLI](../utilities/rav1ator-cli.mdx)，可以根據用戶輸入生成 Av1an 命令。

[GitHub 連結](https://github.com/gianni-rosato/av1an-command-gen/)

## 描述

Av1an Command Generator 專為初學者設計，但經驗豐富的開發人員也可以使用它來加速 Av1an 腳本的生成。該程式生成一個 AV1 影片編碼命令，用於與 [Av1an](../utilities/av1an.mdx) 配合使用，Av1an 是一個分塊影片編碼工具，可以與 [aomenc](../encoders/aomenc.mdx)、[SVT-AV1](../encoders/SVT-AV1.mdx) 和 [rav1e](../encoders/rav1e.mdx) 一起使用。

該工具接受影片分辨率、幀率、所需編碼器、速度預設和目標位元率範圍作為命令行參數。根據這些參數，它計算設定如瓦片列/行、延遲幀數、CRF 和編碼器速度預設。然後，將這些設定注入生成的編碼命令字符串中。

輸出是一個完整的 `av1an` 命令，可以根據指定的設定對影片進行編碼。

## 使用方法

```bash
av1an-command-gen [width] [height] [fps] [encoder] [speed] [bitrate_target]
```

- `width` - 輸入影片的寬度（像素）
- `height` - 輸入影片的高度（像素）
- `fps` - 輸入影片的幀率
- `encoder` - `aom`、`svt` 或 `rav1e`
- `speed` - `slower`、`slow`、`med`、`fast`、`faster`
- `bitrate_target` - `lowest`、`low`、`med`、`high`

## 範例

生成一個用 `rav1e` 以 'med' 速度和 'low' 位元率目標編碼 1280x720 像素、24 fps 的影片的命令：

```bash
av1an-command-gen 1280 720 24 rav1e med low
```

生成一個用 `svt-av1` 以 'fast' 速度和 'high' 位元率目標編碼 1920x1080 像素、30 fps 的影片的命令：

```bash
av1an-command-gen 1920 1080 30 svt fast high
```

## 安裝

此程式需要 [Zig](https://ziglang.org/) v0.11.0 。

構建命令：

```bash
zig build
```

這將生成一個獨立的二進制檔案 `av1an-command-gen`，
位於 `zig-out/bin/` 目錄中。
