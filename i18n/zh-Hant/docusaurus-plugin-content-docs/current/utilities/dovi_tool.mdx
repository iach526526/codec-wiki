# dovi_tool

**dovi_tool** 是一個用 Rust 編寫的命令行工具，結合了多個用於處理 Dolby Vision 的實用工具。

## 安裝

<Tabs>
    <TabItem value="unixlike" label="Linux 與 macOS">
        你可以從 [GitHub Releases](https://github.com/quietvoid/dovi_tool/releases) 下載官方預建的 Linux 和 macOS 二進位檔，或者根據以下說明自行編譯工具。

        確保你已安裝 Rust。

        ```bash
        git clone https://github.com/quietvoid/dovi_tool.git
        cd dovi_tool
        RUSTFLAGS="-C target-cpu=native" cargo build --release
        ```
    </TabItem>
    <TabItem value="windows" label="Windows">
        你可以從 [GitHub Releases](https://github.com/quietvoid/dovi_tool/releases) 下載官方預建的 Windows 二進位檔，或者根據以下說明自行編譯工具。

        確保你已安裝 Rust。

        ```pwsh
        git clone https://github.com/quietvoid/dovi_tool.git
        cd dovi_tool
        set RUSTFLAGS=-C target-cpu=native 
        cargo build --release
        ```
    </TabItem>
</Tabs>

## 使用

```bash title="一般使用"
dovi_tool [OPTIONS] <SUBCOMMAND>
```

```bash title="列出更多參數列表"
dovi_tool <SUBCOMMAND> --help
```
