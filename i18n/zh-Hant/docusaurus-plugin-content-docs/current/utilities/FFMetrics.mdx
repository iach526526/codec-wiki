# FFMetrics

<br/><br/>

FFMetrics 是一款僅限 Windows 的專有圖形使用介面（GUI）工具，用於計算和視覺化影片品質指標結果，並以圖表方式顯示。可選擇的指標包括 PSNR、SSIM 和 VMAF。

## 安裝

:::caution
使用此工具前，你需要確保 FFmpeg 已經在系統 PATH 中設定，並且 VMAF 計算支援需要啟用 `--enable-libvmaf`。 
:::

從他們的 GitHub 主頁 [releases](https://github.com/fifonik/FFMetrics/releases) 下載程式，選擇你想下載的版本，然後將其解壓到你選擇的檔案夾中。
執行 exe 檔案以啟動程式。

## 使用方法

使用「Browse」按鈕插入你的參考（源）影片，使用「Add files」添加你的失真（編碼）影片。選擇你希望執行的指標計算，並選中相應的復選框。
可以通過「VMAF options」選擇自定義 VMAF 模型，完成後點擊「Start」，程式將自動生成每個指標的實時圖表。

根據你的 CPU 性能，此過程可能需要一些時間。
