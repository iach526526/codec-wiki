# rAV1ator

:::info 內容維護中
本條目內容尚未完成，正在進行補充。
:::

rAV1ator 是一個源自 [Aviator](../utilities/Aviator.mdx) 的分支版本，主要是採用 [Av1an](../utilities/av1an.mdx) 和 [rav1e](../encoders/rav1e.mdx) 替代 [SVT-AV1](../encoders/SVT-AV1.mdx) 來進行編碼。這個版本是以 Flatpak 的形式發佈，包含了非 Flathub 的依賴項，你可以透過以下命令來安裝：

```bash
flatpak --user remote-add --no-gpg-verify project-volo https://giannirosato.com/repo && flatpak --user install project-volo net.natesales.rAV1ator
```

Aviator 原本計劃使用 Av1an + rav1e 來取代目前使用的 SVT-AV1，這也是現在 rAV1ator 的後端配置。但後來由於這兩個專案的目標不同，以及 Flathub 在處理 Rust 依賴項時遇到困難，決定將它們分開。Aviator 旨在提供一個簡單且易於使用的 AV1 編碼工具，而 rAV1ator 則針對有興趣深入瞭解編解碼器的愛好者，提供更方便的 Av1an 和 rav1e 訪問。rAV1ator 由 Gianni Rosato 和 Trix 負責維護。
