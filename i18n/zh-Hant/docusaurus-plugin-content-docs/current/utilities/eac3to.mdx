# eac3to

eac3to 是一款由 madshi 開發的命令行工具，主要用於音頻轉換以及處理原始、未加密的藍光光碟（BDMV）。

:::info 兼容性
此軟體僅限於 Windows 系統，但也可以在 Wine 上執行。
:::

:::caution 避免使用
除非被古老的 [私人追蹤器](https://wiki.installgentoo.com/wiki/Private_trackers) 所要求，否則完全沒有必要使用此軟體，因為它需要的眾多依賴並未隨下載檔案一起打包。可以選擇使用如 FFmpeg 之類的其他工具。
:::

## 安裝

- 從 [VideoHelp](https://www.videohelp.com/software/eac3to) 下載，將其拖放到指定檔案夾中，必要時將其添加到 PATH 中。

## 使用方法

### 音頻轉換

範例：
```bash title="將 PCM 音頻轉換為 FLAC"
eac3to source.pcm destination.flac
```

```bash title="將 TrueHD 轉換為 FLAC 同時提取 AC-3 兼容層"
eac3to source.thd destination.flac destination.ac3
```

### 獲取 BDMV 訊息

```bash
eac3to Movie.2024.Bluray/BDMV
```

### 解複製

```bash
eac3to Movie.2024.Bluray/BDMV -demux
```

### 調整音頻延遲
```bash
eac3to input.eac3 output.eac3 1000ms
```
